## 使用说明

1. **安装依赖**
   ```bash
   pip install -r requirements_website.txt
   ```

2. **运行应用**
   ```bash
   python app.py
   ```

3. **访问网站**
   - 首页: http://localhost:5000
   - 产品中心: http://localhost:5000/products
   - 公司简介: http://localhost:5000/about
   - 联系我们: http://localhost:5000/contact
   - 管理后台: http://localhost:5000/admin

4. **测试表单**
   - 访问联系页面
   - 填写并提交表单
   - 查看管理后台确认数据已保存

## 常见问题

### 端口被占用
修改 `app.py` 中的端口号：
```python
app.run(debug=True, host='0.0.0.0', port=5001)
```

### 数据库问题
删除 `contacts.db` 文件，重新运行应用。

### 图片不显示
检查 `static/images/` 目录中是否有对应的图片文件。

### 样式不生效
检查浏览器缓存，或使用硬刷新（Ctrl+F5）。
